# react-netease-cloud-music

仿网易云音乐PC端Web版，练手项目，react + redux + redux-saga

#### API

接口使用[网易云音乐 NodeJS 版 API](https://binaryify.github.io/NeteaseCloudMusicApi/#/)

#### Build Setup

开发：npm run start
